## Markov Model And Bayesian Learning
Papier–Kamień–Nożyce  z uczeniem bayesowskim i ukrytym modelem Markowa

Symulacja gry Papier–Kamień–Nożyce w Pythonie, 
w której jeden gracz stosuje strategię probabilistyczną (łańcuch Markova), 
a drugi gracz z czasem uczy się tej strategii i ją przewiduje (uczenie Baysowskie).
